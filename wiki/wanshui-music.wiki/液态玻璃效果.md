# 液态玻璃效果

> 相关文档：[[架构概览]] | [[页面与组件]] | [[状态管理]]

项目实现了 iOS 26 风格的 Liquid Glass 视觉效果，基于 **SVG 滤镜 + Canvas 离屏渲染 + 物理折射模型**。

## 整体架构

```
用户交互
  │
  ├─ LiquidGlassControls（参数面板）
  │     └─ 修改 GlassParams
  │
  └─ GlassParamsProvider（Context）
        └─ 传递 params → MiniPlayer 等组件
              │
              └─ LiquidGlassBox
                    │
                    ├─ buildLiquidGlassFilter()  ← 生成 SVG 滤镜
                    │     ├─ calculateRefractionProfile()  ← 物理折射计算
                    │     ├─ generateDisplacementMap()      ← Canvas 位移贴图
                    │     └─ generateSpecularMap()          ← Canvas 高光贴图
                    │
                    ├─ installFilter()  ← 注入 DOM
                    └─ 渲染四层结构
```

## 四层渲染结构

`LiquidGlassBox` 组件采用四层叠加实现玻璃效果：

| 层级 | 名称 | 实现 | 作用 |
|------|------|------|------|
| Layer 0 | 基础模糊层 | `backdrop-filter: blur() saturate()` | 背景虚化与饱和度增强 |
| Layer 1 | SVG 折射层 | `backdrop-filter: url(#filterId)` | 物理折射扭曲 + 高光 |
| Layer 3 | 边缘层 | `box-shadow: inset ...` | 顶部高光线 + 内阴影 + 边框 |
| Layer 4 | 内容层 | `z-index: 4` | 实际子内容 |

> 注意：没有 Layer 2，编号直接从 1 跳到 3，与 iOS 的层级命名一致。

## 物理折射模型

### 表面函数

玻璃表面形状由数学函数定义：

```typescript
SURFACE_FNS = {
  convex_squircle: (x) => (1 - (1-x)^4)^0.25,   // 凸面 Squircle
  convex_circle:   (x) => sqrt(1 - (1-x)^2),      // 凸面圆形
  concave:         (x) => 1 - sqrt(1 - (1-x)^2),  // 凹面
  lip:             (x) => lerp(convex, concave, smoothstep(x)),  // 唇形过渡
}
```

默认使用 `convex_squircle`，模拟 Apple 的 Squircle 圆角。

### 折射计算

`calculateRefractionProfile()` 基于 Snell 定律计算折射偏移：

1. 对玻璃边缘的每个采样点（默认 128 个），计算表面法线
2. 根据折射率（IOR）计算折射方向
3. 投射到玻璃底部，得到位移量

```
入射光 → 玻璃表面 → 折射 → 位移
         ↑ 法线       ↓ IOR
         └── Snell 定律 ──┘
```

### 形状支持

| 形状 | 说明 |
|------|------|
| `squircle` | 标准 Squircle 圆角矩形 |
| `superellipse` | 超椭圆（`|x/r|^n + |y/r|^n = 1`），n=5 近似 Apple 圆角 |

超椭圆通过 `superellipseDist()` 计算有符号距离，用于确定边缘区域。

## Canvas 贴图生成

### 位移贴图（Displacement Map）

`generateDisplacementMap()` 在离屏 Canvas 上生成 R/G 通道位移图：

- 遍历每个像素，计算到边缘的距离
- 根据折射 profile 查找位移量
- 将位移方向编码到 R（X 方向）和 G（Y 方向）通道
- 中心值 128，偏移范围 ±127

### 高光贴图（Specular Map）

`generateSpecularMap()` 生成光照高光图：

- 模拟从左上方（π/3 角度）照射的光源
- 计算每个边缘像素的法线与光源方向的点积
- 边缘区域产生高光，中心区域保持暗色

## SVG 滤镜管线

`buildLiquidGlassFilter()` 生成的 SVG 滤镜包含 10 个步骤：

```
SourceGraphic
  │
  ├─ 1. feGaussianBlur          → 背景模糊
  │
  ├─ 2. feImage (位移贴图)      → 加载 Canvas 生成的贴图
  ├─ 3. feDisplacementMap       → 应用折射扭曲
  │
  ├─ 4. feColorMatrix (saturate)→ 饱和度增强
  │
  ├─ 5. feImage (高光贴图)      → 加载高光层
  ├─ 6. feComposite             → 用高光遮罩饱和度层
  ├─ 7. feComponentTransfer     → 降低高光透明度
  │
  ├─ 8. feBlend                 → 合并饱和度增强
  ├─ 9. feBlend (screen)        → 叠加白色高光
  │
  ├─ 10. feSpecularLighting     → 顶部镜面光照（增加厚度感）
  └─ feBlend (screen)           → 最终合成
```

### Fallback 滤镜

当 Canvas 生成失败时，使用 `buildSimpleLiquidGlassFilter()` 作为降级方案：

- 使用 `feTurbulence` 生成噪声代替位移贴图
- 效果较弱但保证可用
- 不依赖 Canvas API

## 效果模式

| 模式 | 模糊量 | 高光透明度 | 饱和度 | 背景模糊 |
|------|--------|-----------|--------|---------|
| `regular` | 原始值 | 原始值 | 原始值 | 24px / 1.6 |
| `clear` | ×0.6 | ×0.5 | ×0.7 | 16px / 1.3 |
| `none` | 0 | 0 | 0 | 无 |

## DOM 管理

`liquidGlassManager.ts` 负责管理全局 SVG `<defs>` 容器：

- 在 `<body>` 中创建隐藏的 `<svg>` 元素（`#liquid-glass-svg-root`）
- 所有滤镜定义作为 `<filter>` 子元素插入
- 使用 `DOMParser` 解析 SVG 片段，确保 namespace 正确
- 每个组件实例有唯一 filterId，卸载时自动清理

## 参数控制面板

`LiquidGlassControls` 提供可视化参数调节界面：

- **形状**：圆角半径、超椭圆指数
- **折射**：玻璃厚度、边框宽度、折射率、缩放比
- **外观**：模糊量、高光透明度、高光饱和度
- **阴影**：内阴影模糊/扩散、外阴影模糊
- **着色**：透明度、RGB 分量

参数通过 [[状态管理|GlassParams Context]] 在组件间共享。

## 性能优化

- **防抖重建**：参数变化后 150ms 延迟，避免滑块拖动时频繁重建
- **ResizeObserver**：仅在元素尺寸变化时重建滤镜
- **Canvas 缓存**：位移贴图和高光贴图以 `data:image/png` 形式内嵌到 SVG
- **旧滤镜清理**：新滤镜安装后自动移除旧滤镜，避免 DOM 膨胀
