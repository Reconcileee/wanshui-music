## 1. 产品概述

一个纯浏览器端的音频转字幕工具，用户上传音频文件后自动识别语音内容并生成带时间戳的 LRC（歌词）和 SRT（字幕）文件，支持多语言识别。所有处理在本地浏览器完成，无需上传到服务器，保护用户隐私。

- 解决问题：用户需要快速将音频（如歌曲、播客、演讲）转换为带时间轴的歌词/字幕文件
- 目标用户：音乐爱好者、视频创作者、字幕组、播客制作者
- 价值：免费、隐私安全、无需注册、跨平台使用

## 2. 核心功能

### 2.1 功能模块

1. **主页（单页应用）**：音频上传、语言选择、识别进度、结果预览、文件下载

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页 | 顶部导航 | Logo、标题、GitHub 链接 |
| 主页 | 上传区域 | 拖拽/点击上传音频，支持 wav/mp3/mp4/m4a/flac，限制时长 2 小时、大小 200MB |
| 主页 | 语言选择 | 下拉选择源语言：中文、英语、日语、韩语、法语、德语等，含"自动检测" |
| 主页 | 识别控制 | 开始识别按钮、进度条、实时状态文字（加载模型/识别中/完成） |
| 主页 | 结果预览 | 双标签页切换 LRC / SRT 预览，带时间戳的行级展示 |
| 主页 | 下载区 | 分别下载 .lrc 和 .srt 文件，复制内容按钮 |
| 主页 | 模型选择 | 可选 Whisper 模型大小：tiny（快）/ base（均衡）/ small（精准） |

## 3. 核心流程

用户打开页面 → 上传音频文件 → 选择源语言和模型大小 → 点击"生成字幕" → 浏览器加载 Whisper 模型 → 实时显示识别进度 → 识别完成后预览 LRC/SRT 内容 → 下载所需格式文件

```mermaid
    "C --> D[点击生成字幕]"
    "D --> E{模型是否已加载}"
    "E -->|否| F[下载并加载 Whisper 模型]"
    "E -->|是| G[开始语音识别]"
    "F --> G"
    "G --> H[实时显示进度]"
    "H --> I{识别完成?}"
    "I -->|否| H"
    "I -->|是| J[生成 LRC 和 SRT]"
    "J --> K[预览结果]"
    "K --> L[下载文件]"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：iOS 26 液态玻璃（Liquid Glass）风格，深色背景为主
- **主色调**：深空灰背景 (#0a0a0c) + 玻璃半透明面板 + 苹果蓝 (#0a84ff) 强调色
- **按钮风格**：圆角胶囊形 (border-radius: 999px)，主按钮填充苹果蓝，次按钮玻璃质感
- **字体**：SF Pro Display / -apple-system / PingFang SC，标题 28px bold，正文 15px
- **布局**：居中卡片式布局，最大宽度 720px，垂直堆叠
- **图标**：Lucide React 线性图标，1.5px 描边
- **动效**：上传区拖拽高亮、按钮按压回弹、进度条流光、结果淡入上滑、玻璃面板悬浮感

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-----------|-------------|
| 主页 | 顶部 | 居中标题"音频转字幕"，副标题说明，毛玻璃导航栏 |
| 主页 | 上传区 | 虚线边框圆角卡片，拖拽时蓝色高光，中央上传图标+文字，支持格式说明 |
| 主页 | 配置区 | 玻璃面板内含语言下拉、模型选择段控件、文件信息展示 |
| 主页 | 进度区 | 环形进度指示器 + 状态文字 + 取消按钮 |
| 主页 | 结果区 | 标签页切换 LRC/SRT，等宽字体预览，右上角下载/复制按钮 |
| 主页 | 背景 | 多层渐变光晕（蓝/紫/粉），缓慢浮动动画，营造氛围 |

### 4.3 响应式

- 桌面优先（720px 居中卡片）
- 移动端自适应：卡片宽度 100%，内边距缩小，按钮全宽
- 触摸优化：上传区点击区域 ≥ 48px，按钮高度 ≥ 44px
